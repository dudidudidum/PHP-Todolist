# Bài tập PHP - Ứng dụng To-Do List

**Họ và tên:** Tạ Đức Dũng
**Mã Sinh Viên:** 2205HTTC016


---

### Mô tả
Đây là bài tập xây dựng ứng dụng quản lý công việc (To-Do List) bằng PHP thuần, MySQL và Bootstrap 5.

### Chức năng chính
* Xác thực người dùng (Đăng ký, Đăng nhập, Đăng xuất).
* Quản lý công việc (Thêm, Xem, Sửa, Xóa).
* Lọc và sắp xếp công việc.
* Giao diện responsive dùng Bootstrap.

*Ghi chú: Tệp db.php (chứa thông tin kết nối CSDL) đã được ẩn đi bằng .gitignore vì lý do bảo mật.*
